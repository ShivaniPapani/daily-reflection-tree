# Daily Reflection Decision Tree

## Objective
Create a deterministic decision tree that helps users reflect on their daily productivity and identify improvements.

## Decision Flow

START

Q1: Did you complete your main goal today?

If YES:
    Q2: What helped you succeed?
        - Good planning
        - Focus
        - Support from others

   

If NO:
    Q3: What was the main obstacle?
        - Lack of time
        - Distractions
        - Unclear task

    

Q4: What is one improvement you will apply tomorrow?

END
