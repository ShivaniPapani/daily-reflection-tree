# Daily Reflection Decision Tree

## Overview
This project implements a deterministic decision tree that helps users reflect on their daily productivity and identify improvements.

## Features
- Rule-based decision tree
- Daily reflection prompts
- Improvement suggestions
- Optional AI reflection assistant

## Approach

1. Defined a structured reflection framework.
2. Designed a deterministic decision tree.
3. Implemented the logic using Python conditional statements.
4. Added an optional AI reflection assistant.

## Guardrails Against AI Hallucination

To prevent hallucinations:
- Core decision logic is deterministic.
- AI suggestions are rule-based.
- Prompts are constrained and structured.
- Output recommendations are practical and task-focused.

## How to Run

Run the deterministic reflection system:
python reflection_tree.py

Run the AI reflection assistant:

python ai_agent.py

## Future Improvements
